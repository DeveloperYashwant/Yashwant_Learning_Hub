# üìö **SQL Functions Introduction**

**SQL Functions** are **ready-made formulas** that perform operations on data stored in tables.

- **Why use functions?**  
  üëâ To **modify**, **analyze**, or **format** the data easily.

---

# 1. **String Functions**

‚û°Ô∏è These functions **work with text (strings)**.

| Function | Use | Example |
|:---------|:----|:--------|
| `UPPER()` | Converts text to **uppercase** | `SELECT UPPER('hello');` ‚Üí **HELLO** |
| `LOWER()` | Converts text to **lowercase** | `SELECT LOWER('HELLO');` ‚Üí **hello** |
| `LENGTH()` | Finds **length of text** | `SELECT LENGTH('Hello');` ‚Üí **5** |
| `SUBSTRING()` | Extracts a part of text | `SELECT SUBSTRING('HelloWorld',1,5);` ‚Üí **Hello** |
| `CONCAT()` | Joins two strings | `SELECT CONCAT('Hello', 'World');` ‚Üí **HelloWorld** |
| `TRIM()` | Removes extra spaces | `SELECT TRIM(' Hello ');` ‚Üí **Hello** |

‚úÖ **Example in Table:**
```sql
SELECT UPPER(Name) FROM Students;
```

---

# 2. **Numeric Functions**

‚û°Ô∏è These functions **work with numbers**.

| Function | Use | Example |
|:---------|:----|:--------|
| `ABS()` | Returns **absolute value** | `SELECT ABS(-10);` ‚Üí **10** |
| `ROUND()` | Rounds a number | `SELECT ROUND(12.345, 2);` ‚Üí **12.35** |
| `CEIL()` | Rounds number **up** | `SELECT CEIL(4.2);` ‚Üí **5** |
| `FLOOR()` | Rounds number **down** | `SELECT FLOOR(4.8);` ‚Üí **4** |
| `MOD()` | Finds **remainder** | `SELECT MOD(10,3);` ‚Üí **1** |
| `POWER()` | Raises to power | `SELECT POWER(2,3);` ‚Üí **8** |

‚úÖ **Example in Table:**
```sql
SELECT ROUND(Marks, 0) FROM Students;
```

---

# 3. **Date Functions**

‚û°Ô∏è These functions **work with dates and times**.

| Function | Use | Example |
|:---------|:----|:--------|
| `CURRENT_DATE` | Gives today's **date** | `SELECT CURRENT_DATE;` |
| `CURRENT_TIME` | Gives current **time** | `SELECT CURRENT_TIME;` |
| `NOW()` | Gives current **date & time** | `SELECT NOW();` |
| `YEAR()` | Extracts **year** | `SELECT YEAR('2025-04-29');` ‚Üí **2025** |
| `MONTH()` | Extracts **month** | `SELECT MONTH('2025-04-29');` ‚Üí **4** |
| `DAY()` | Extracts **day** | `SELECT DAY('2025-04-29');` ‚Üí **29** |

‚úÖ **Example in Table:**
```sql
SELECT YEAR(DateOfBirth) FROM Students;
```

---

# 4. **Combined Practical Example**

```sql
SELECT 
  UPPER(Name) AS NameInCaps,
  ROUND(Marks, 1) AS RoundedMarks,
  YEAR(DateOfBirth) AS BirthYear
FROM Students;


---

# 5. **Important Points**

- Functions are used inside the **SELECT** statement.
- Functions **do not change** the original data unless used with **UPDATE**.
- You can **combine** multiple functions together!

Example:
```sql
SELECT UPPER(SUBSTRING(Name, 1, 3)) FROM Students;
```
